顔認証対話型案内システム

2019年の大学祭で提示した案内システムです。

URL

Github

担当

サーバーサイド

プロダクトの目的

大学祭のサークル展示のため

ターゲット

大学祭に来た人

機能

  • 顔がカメラに入った場合会話を開始する
  • カメラに顔が入っているかチェックする
  • フロント側でブラウザの音声認識機能を利用して音声データをテキストデータに変換する
  • 発話された単語でjsonデータから発話内容を選び、サークルの紹介などのテキストをフロントに送る
  • フロント側でブラウザの音声生成機能を利用してテキストデータを音声データに変換し発話する
