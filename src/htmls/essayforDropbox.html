essayForDropbox

url指定した論文をpdfとしてDropboxに保存するプロダクトです。

URL

Github

Qiita

担当

自作プロダクト

サイトの目的

大きい論文をモバイル回線でダウンロードしたくない問題や、
ブラウザに保存してある論文urlが積みあがっていく問題を解決する

機能

  • pdfのurlを受け取りダウンロードする
  • ダウンロードされたpdfをDropboxにアップロードする

コーディングについて

以下のものを用いてAPIサーバを構築しました。

  • Heroku
  • Go
  • Gin
  • Dropbox
